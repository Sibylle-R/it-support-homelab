# IT Support Homelab

Dieses Repository dokumentiert mein persönliches IT-Homelab.

Das Ziel dieses Labs ist es, typische Tätigkeiten aus dem 1st-Level- und Junior-IT-Support praktisch zu üben und besser zu verstehen.

Ich habe dieses Homelab erstellt bevor ich mit meiner Ausbildung als Informatiker richtung Systemtechnik gestartet habe, um Fehler zu machen und diese selbst zu beheben.

---

## Inhalte & Schwerpunkte

- Windows Server Grundsetup
- Active Directory Grundlagen
- Benutzer- und Gruppenverwaltung
- Gruppenrichtlinien (GPOs)
- Client-Domain-Join
- DNS-Grundlagen
- Typische Troubleshooting-Szenarien 

---

## Umgebung

- **Host OS:** Windows 11 Pro  
- **Virtualisierung:** Hyper-V  
- **Server:** Windows Server 2022 Standard (Desktop Experience)  
- **Client:** Windows 11  
- **Netzwerk:** Interner Hyper-V Switch  
- **Domain:** `lab.local`

---

## Projektübersicht

### Projekt 01 – Active Directory Grundsetup
Einrichtung eines Windows Servers als Domain Controller inklusive DNS und Domain-Erstellung.

### Projekt 02 – Benutzer- & Gruppenverwaltung
Erstellung von Benutzern und Sicherheitsgruppen sowie Tests mit Gruppenmitgliedschaften.

### Projekt 03 – Gruppenrichtlinien (GPO)
Erstellung und Anwendung von Gruppenrichtlinien zur zentralen Steuerung von Benutzereinstellungen.

### Projekt 04 – Troubleshooting
Analyse und Behebung typischer Probleme wie DNS-Fehler, fehlgeschlagene Domain-Joins oder nicht greifende GPOs.

### Projekt 05 – Fileserver & Berechtigungen
Einrichtung eines einfachen Fileservers mit NTFS-Berechtigungen basierend auf Gruppen.
