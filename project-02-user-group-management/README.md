# Projekt 02 – Benutzer- und Gruppenverwaltung

## Ziel
IT-Support-Aufgaben rund um Benutzer und Gruppen üben.

## Umsetzung
- Testbenutzer in Active Directory erstellt
- Sicherheitsgruppe (z. B. IT-Support) angelegt
- Benutzer der Gruppe hinzugefügt
- Anmeldung mit Domänenbenutzer getestet

## Ergebnis
Grundlegende Benutzer- und Gruppenverwaltung in Active Directory erfolgreich umgesetzt.
