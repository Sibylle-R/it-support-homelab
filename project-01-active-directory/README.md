# Projekt 01 – Active Directory Grundsetup

## Ziel
Aufbau eines einfachen Windows-Domainsystems, um die Grundlagen von Active Directory zu verstehen.

## Umsetzung
- Windows Server 2022 als Domain Controller (DC01) installiert
- Neue Domain `lab.local` erstellt
- DNS-Rolle eingerichtet
- Server erfolgreich zum Domain Controller promotet

## Ergebnis
Active Directory ist funktionsfähig und bildet die Basis für Benutzer-, Gruppen- und Richtlinienverwaltung.
