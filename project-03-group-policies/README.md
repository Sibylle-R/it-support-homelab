# Projekt 03 – Gruppenrichtlinien (GPO)

## Ziel
Verstehen, wie Gruppenrichtlinien eingesetzt werden, um Benutzer- oder Systemeinstellungen zentral zu steuern.

## Umsetzung
- Neue Gruppenrichtlinie erstellt
- Zugriff auf die Systemsteuerung eingeschränkt
- GPO mit Organisationseinheit verknüpft
- Wirkung auf dem Client überprüft

## Ergebnis
Gruppenrichtlinien greifen korrekt und beeinflussen das Benutzerverhalten wie erwartet.
